# FAMプランセクション デザインドキュメント

## 概要

個人アスリート向けに、FAMの食事プランを2つのセクションで訴求する。
定期便を「主役」として先に強く訴求し、その下でACTIVE単品を比較選択できる構成。

## 構成

```
セクション1: 定期便プラン（ダーク＆モノトーン）
  ↓
セクション2: ACTIVEプラン比較（FAMオレンジ + フィルター + カード）
```

---

## セクション1: 定期便プラン

### デザイン方針
- 背景: #1B1D1A（ダーク）
- 高級感・プレミアム感を演出（Nike, Apple的）
- 定期便の価値を「悩み → 解決策」で伝える

### PC版レイアウト（高さ720px）
- 2カラム: 左テキスト(50%) + 右料理写真(50%)
- 1440px max-width、左64px余白

### 左カラム内容（上から）
1. キッカー: "Subscription Plan" (Poppins 12px #fff opacity 0.5)
2. タイトル: "何を食べればいいか、もう迷わない。" (Hiragino Sans W6 40px #fff)
3. 説明文: 悩み + 解決策 (Hiragino Sans W4 16px #fff opacity 0.7)
4. 価格: "初回 50% OFF ¥950〜/1食" (#FB4C15アクセント)
5. CTA: "定期便を見る →" (白枠ボタン)

### 右カラム
- 料理写真（大きく・高品質）
- object-fit: cover

### SP版
- 縦積み: 上に料理写真(280px) → 下にテキスト+価格+CTA
- 左右24px余白

### Shopifyカスタマイズ項目
- キッカーテキスト
- タイトル（改行可）
- 説明文（改行可）
- 価格表示テキスト
- 割引テキスト
- CTA テキスト / URL
- 料理写真（image_picker）

---

## セクション2: ACTIVEプラン比較

### デザイン方針
- 背景: #FB4C15（FAMオレンジ）
- カード4枚で全ACTIVEプランを比較
- フィルターボタンで目的別に絞り込み

### PC版レイアウト
- 1440px max-width、左64px余白
- 上64px余白 → タイトル → フィルター → カード4枚横並び → 64px下余白

### タイトル
- "まずは試してみたい方へ" (Hiragino Sans W6 40px #1B1D1A)

### フィルターボタン
- [すべて] [筋力アップ] [維持] [減量]
- 選択中: 白背景 + #1B1D1A文字
- 未選択: 透明背景 + 白枠 + 白文字

### カード仕様（白背景・角R 0・シャドウ）
各カード内容（上から）:
1. 料理写真 (アスペクト比 4:3)
2. プラン名: "ACTIVE 60" (Poppins Medium)
3. ターゲット: "70kg以上 / 筋力アップ" (Hiragino Sans)
4. たんぱく質: "P60g"
5. 価格: "¥1,800/食"
6. CTA: "詳しく見る →"

### SP版
- カルーセル: 1枚 + 0.3枚チラ見え
- スワイプで切り替え（スナップスクロール）
- 左24px余白

### フィルターロジック
各プランのカテゴリ:
- ACTIVE 60: 筋力アップ
- ACTIVE 50: 筋力アップ, 維持
- ACTIVE 40: 維持, 筋力アップ
- ACTIVE 30: 維持, 減量

フィルター選択時: 該当しないカードをopacity 0.3 + blur(2px)にする（非表示ではなく薄くする）

### Shopifyカスタマイズ項目
- セクションタイトル
- 背景色
- ブロック（カード）× 最大6枚:
  - プラン名
  - ターゲット説明
  - 目的カテゴリ（select: 筋力アップ/維持/減量）
  - たんぱく質量
  - 価格テキスト
  - 料理写真
  - CTA テキスト / URL

## テーマCSS干渉対策（共通）

- フォントは `!important` で上書き
